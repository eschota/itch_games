## Long Description

`Orbital Courier is a compact browser arcade run about reading the lane,
collecting cyan energy cores, and surviving red debris for a higher score. The
run starts immediately, the ship responds to keyboard, mouse, and touch input,
and the latest build adds procedural Web Audio feedback for starts, engine
motion, pickups, hits, near misses, and run completion.`

## Controls

Keyboard:
`A` / `D` or left / right arrows move the courier. `Space` restarts after a run
ends.

Mouse and touch:
Drag on the play field to steer. Click or tap after a run ends to restart.

Audio:
Browser audio starts after the first keyboard, mouse, or touch gesture.
